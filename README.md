This project contains a focused implementation of the **Home**, **Posts index**, and **per-post pages** using Astro and Tailwind.

## How to run
1. `npm install`
2. `npm run dev`
3. Visit `http://localhost:3000`

## Where to add posts
Add Markdown files under `src/content/posts/` with frontmatter including `title`, `date`, `description`, and `slug`.
